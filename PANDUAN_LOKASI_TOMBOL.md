# Dimana Tombol "Get Code" / Download?

Tergantung Anda sedang berada di halaman mana:

## 1. Jika Anda di Editor Ini (AI Studio)
Anda ingin **mengambil file aplikasi** yang sudah saya buat untuk disimpan di komputer?

*   **Cari di Pojok Kanan Atas**: Lihat ikon **Panah ke Bawah (Download)** atau tombol bertuliskan **Export**.
*   Pilih **Download as ZIP**.
*   *Jika tidak ada tombol tersebut*: Anda mungkin perlu menyalin file secara manual, atau mencari menu **"Open in..."** untuk membukanya di editor lain lalu download dari sana.

---

## 2. Jika Anda di Website GitHub
Anda sedang melihat halaman repository yang baru dibuat?

*   **Tombol Hijau "<> Code"**:
    *   **Letak**: Di sebelah kanan atas daftar file.
    *   **Fungsi**: Untuk **MENDOWNLOAD** kode dari GitHub ke komputer Anda (Clone/Download ZIP).
    *   **PENTING**: Tombol ini **BUKAN untuk Upload**. Jangan gunakan tombol ini jika Anda ingin memasukkan kode ke GitHub.

*   **Tombol untuk Upload (Memasukkan Kode)**:
    *   **Jika Repository Kosong**: Cari link kecil berwarna biru di tengah layar bertuliskan **"uploading an existing file"**.
    *   **Jika Repository Sudah Ada Isi**: Klik tombol **"Add file"** (di sebelah kiri tombol hijau Code) -> Pilih **"Upload files"**.

---

### Ringkasan Langkah (Metode Paling Mudah):
1.  **Di Editor Ini**: Cari tombol **Download/Export** -> Simpan file ZIP -> Ekstrak folder.
2.  **Di GitHub**: Cari link **"uploading an existing file"** -> Tarik (Drag & Drop) semua file dari folder tadi ke browser.
