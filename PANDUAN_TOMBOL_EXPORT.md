# Mencari Tombol Download / Export di Google AI Studio

Karena tampilan Google AI Studio bisa berbeda-beda tergantung update, coba cari di lokasi berikut:

## 1. Pojok Kanan Atas (Header)
Lihat ke bagian paling atas layar (di baris yang sama dengan logo Google AI Studio). Cari tombol-tombol ini:

*   **Tombol "Export"**: Biasanya berupa teks atau ikon panah keluar.
*   **Ikon "< >" (Code)**: Kadang tombol ini membuka tampilan kode yang bisa di-copy.
*   **Menu Titik Tiga (...)**: Klik titik tiga di pojok kanan atas, lalu cari menu **Download** atau **Export to...**.

## 2. Jika Tidak Menemukan Tombol Download
Jika Anda benar-benar tidak bisa menemukan tombol untuk mendownload semua file sekaligus, Anda bisa melakukan cara manual (Copy-Paste):

1.  Saya bisa menampilkan isi file satu per satu di sini.
2.  Anda copy isinya.
3.  Anda buat file baru di komputer Anda (menggunakan Notepad) dan paste isinya.
4.  Simpan dengan nama yang sesuai (misal: `App.tsx`).

**Apakah Anda ingin saya tampilkan kode lengkapnya di sini agar bisa di-copy manual?** (Jawab "Ya" jika mau).

---

## Alternatif: Menggunakan GitHub Desktop (Jika Paham)
Jika Anda sudah menghubungkan akun GitHub di pengaturan AI Studio, kadang ada tombol **"Push to GitHub"** langsung. Coba cari ikon logo GitHub (kucing gurita) di toolbar.
