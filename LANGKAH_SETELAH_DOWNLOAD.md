# Langkah Setelah Download ZIP

Bagus! Anda sudah memiliki file proyeknya. Sekarang ikuti langkah-langkah santai ini untuk menaruhnya di internet (online).

## Langkah 1: Buka File ZIP (Ekstrak)
1.  Cari file ZIP yang baru saja Anda download di komputer (biasanya di folder **Downloads**).
2.  Klik kanan file tersebut -> Pilih **Extract All** (atau **Ekstrak Semua**).
3.  Ingat lokasi folder hasil ekstrak tersebut.

## Langkah 2: Buat "Rumah" di GitHub
1.  Buka browser dan login ke [github.com](https://github.com).
2.  Di pojok kanan atas, klik tanda **+** lalu pilih **New repository**.
3.  **Repository name**: Isi dengan nama aplikasi, misal `spensa-digital`.
4.  **Public/Private**: Pilih **Public** (agar Vercel yang gratisan bisa membacanya).
5.  **JANGAN** centang "Add a README file" (biarkan kosong).
6.  Klik tombol hijau **Create repository**.

## Langkah 3: Upload File ke GitHub (Cara Paling Mudah)
Setelah repository jadi, Anda akan melihat halaman kosong dengan banyak kode. Abaikan kodenya.

1.  Cari link kecil berwarna biru di tengah layar bertuliskan: **uploading an existing file**. Klik link itu.
2.  Buka folder hasil ekstrak di komputer Anda (Langkah 1).
3.  **Pilih SEMUA file dan folder** di dalamnya (Ctrl+A).
4.  **Tarik (Drag & Drop)** semua file itu ke browser (ke kotak upload di GitHub).
5.  Tunggu proses upload selesai (bar hijau penuh).
6.  Scroll ke bawah. Di kotak "Commit changes", ketik: `Upload aplikasi pertama`.
7.  Klik tombol hijau **Commit changes**.

*Sekarang kode Anda sudah aman di GitHub!*

## Langkah 4: Online-kan dengan Vercel (Agar Bisa Diakses Orang Lain)
1.  Buka tab baru dan pergi ke [vercel.com](https://vercel.com).
2.  Login menggunakan tombol **Continue with GitHub**.
3.  Di dashboard Vercel, klik tombol **Add New...** -> **Project**.
4.  Di daftar "Import Git Repository", Anda akan melihat `spensa-digital` (atau nama yang Anda buat tadi).
5.  Klik tombol **Import** di sebelahnya.
6.  Di halaman selanjutnya, scroll ke bawah dan langsung klik **Deploy**. (Tidak perlu ubah pengaturan apa-apa).

## Selesai! 🎉
Tunggu sekitar 1-2 menit hingga muncul kembang api di layar.
Klik kotak pratinjau website atau tombol **Visit**.
Link website tersebut (misal: `spensa-digital.vercel.app`) bisa Anda bagikan ke Kepala Sekolah atau Guru lain!
