# Apa itu Akun Git? (Git vs GitHub)

Seringkali ada kebingungan antara **Git** dan **GitHub**. Berikut perbedaannya:

## 1. Git (Alatnya)
*   **Apa itu?**: Software gratis yang diinstall di komputer Anda.
*   **Fungsinya**: Mencatat setiap perubahan pada kode Anda (seperti mesin waktu).
*   **Apakah butuh akun?**: Tidak. Anda hanya perlu mengatur "Nama" dan "Email" di komputer agar Git tahu siapa yang melakukan perubahan. Ini disebut *Git Configuration*.

## 2. GitHub (Websitenya)
*   **Apa itu?**: Website untuk menyimpan kode program secara online (seperti Google Drive khusus programmer).
*   **Fungsinya**:
    *   Menyimpan backup kode Anda.
    *   Memungkinkan orang lain melihat atau membantu kode Anda.
    *   Menghubungkan kode ke layanan hosting (seperti Vercel) agar website bisa online.
*   **Apakah butuh akun?**: **YA**. Ini yang dimaksud dengan "Punya Akun Git" oleh kebanyakan orang.
    *   Daftar gratis di: [github.com](https://github.com)

## Kesimpulan
Saat panduan meminta "Akun Git", maksudnya adalah **Akun GitHub**.

### Langkah yang harus dilakukan:
1.  Buka [github.com/signup](https://github.com/signup).
2.  Daftar dengan email Anda.
3.  Buat username (misal: `guru-smp-1`).
4.  Verifikasi email.
5.  Akun siap digunakan untuk login di terminal atau aplikasi desktop GitHub.
