# Langkah Pertama untuk Deploy (Online-kan Aplikasi)

Langkah paling pertama yang **WAJIB** dilakukan sebelum hal lain adalah:

## 1. AMBIL KODE APLIKASI INI (DOWNLOAD)

Anda tidak bisa men-deploy langsung dari dalam editor AI ini. Anda harus memindahkan kodenya ke komputer Anda atau ke GitHub terlebih dahulu.

### Caranya:
1.  Lihat ke **Pojok Kanan Atas** layar editor ini.
2.  Cari tombol **Export** atau ikon **Panah Bawah (Download)**.
3.  Pilih **Download as ZIP**.
4.  Setelah ter-download, **Ekstrak** (Unzip) file tersebut di komputer Anda.

---

### Setelah itu apa?
Baru lanjut ke langkah kedua: **Upload ke GitHub**.

*(Jika Anda sudah mendownloadnya, abaikan langkah ini dan lanjut ke panduan upload GitHub yang sudah saya buatkan sebelumnya di file `PANDUAN_SANGAT_MUDAH.md`)*.
